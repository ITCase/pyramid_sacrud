Installing `sacrud`
===================

Install from github
-------------------

.. code::

    pip install git+http://github.com/ITCase/sacrud.git

current develop version

.. code::

    pip install git+http://github.com/ITCase/sacrud.git@develop

PyPi
----

.. code::

    pip install sacrud

Source
------

.. code::

    git clone git+http://github.com/ITCase/sacrud.git
    python setup.py install
