Installing
==========

github
------

.. code-block:: bash

    pip install git+http://github.com/ITCase/pyramid_sacrud.git

PyPi
----

.. code-block:: bash

    pip install pyramid_sacrud

source
------

.. code-block:: bash

    git clone git+http://github.com/ITCase/pyramid_sacrud.git
    python setup.py install

contribute
----------

.. code-block:: bash

    git clone git+http://github.com/ITCase/pyramid_sacrud.git
    python setup.py develop
