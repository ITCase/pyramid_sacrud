Installing
==========

GitHub
------

.. code-block:: bash

    pip install git+http://github.com/ITCase/pyramid_sacrud.git

PyPi
----

.. code-block:: bash

    pip install pyramid_sacrud

Source
------

.. code-block:: bash

    git clone git+http://github.com/ITCase/pyramid_sacrud.git
    python setup.py install

Contribute
----------

.. code-block:: bash

    git clone git+http://github.com/ITCase/pyramid_sacrud.git
    python setup.py develop
