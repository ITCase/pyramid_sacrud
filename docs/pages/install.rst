Installing
==========

GitHub
------

.. code-block:: bash

    pip install git+http://github.com/sacrud/pyramid_sacrud.git

PyPi
----

.. code-block:: bash

    pip install pyramid_sacrud

Source
------

.. code-block:: bash

    git clone git+http://github.com/sacrud/pyramid_sacrud.git
    cd pyramid_sacrud
    pip install .

Develop
-------

.. code-block:: bash

    git clone git+http://github.com/sacrud/pyramid_sacrud.git
    cd pyramid_sacrud
    pip install -e .
